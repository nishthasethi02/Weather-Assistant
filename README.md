🌦️ AI Weather Forecaster
An intelligent weather forecasting application that provides real-time weather updates and 5-day forecasts using voice interaction and data visualization.

🚀 Features
🌡️ Get current weather of any city
📊 View 5-day weather forecast
🎤 Voice input support using speech recognition
🔊 Text-to-speech output for weather updates
📈 Visualize weather trends using matplotlib
🌍 Real-time data from OpenWeather API

🛠️ Tech Stack
Python
requests – API calls
pandas – data processing
matplotlib – data visualization
pyttsx3 – text-to-speech
speech_recognition – voice input

🎤 How It Works
The app listens for voice commands:
"current" → current weather
"forecast" → 5-day forecast
"exit" → stop the app
User speaks city name 🎤
App fetches data from API 🌐
Displays results + speaks output 🔊
Shows graphs for forecast 📊

📊 Sample Output
Temperature, humidity, wind speed
Sunrise & sunset time
Weather condition description
Forecast graphs (temperature, humidity, rain probability)
